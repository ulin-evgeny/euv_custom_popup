euv_custom_popup v1.0.0

https://github.com/ulin-evgeny/euv_custom_popup

JQuery-плагин для вызова кроссбраузерного, адаптивного, стилизованного попапа, размещенного посередине по горизонтали экрана (с учетом скроллбаров различных браузеров), который можно скроллить.

Тестировался в браузерах: IE11, Edge 44, Google Chrome 78, Opera 64, FF 70.

Попап без reset_on_close

Обычный попап, содержимое которого не сбрасывается после закрытия. Подойдет для вызова какого-нибудь статического контента.

Закрыть Случайный цвет

Попап c reset_on_close: 'clone'

Попап, содержимое которого сбрасывается за счет JQuery-функции clone.
Здесь может быть проблема: если в обработчиках используется переменная, которая была установлена до вызова попапа, код с ней будет работать только при первом вызове, так как элемента, на который ссылается переменная, в последующих вызовах существовать не будет.
Пример - кнопка "Случайный цвет". При первом открытии попапа она работать будет, при последующих - нет.

Попап c reset_on_close: 'html'

Попап, содержимое которого сбрасывается за счет JQuery-функции html. Для того, чтобы повесить обработчики, рекомендуется использовать callback-функцию при вызове попапа (тогда пользователь не увидит изменение содержимого попапа, т.к. функция вызывается до показа попапа).
Данный вариант лучше всего подходит для попапа, содержимое которого будет меняться (здесь нет таких проблем, которые есть в clone).
